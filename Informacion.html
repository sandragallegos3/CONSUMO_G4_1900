Operaciones que realiza

Pasajeros

El presente formulario "Pasajeros" nos sirve primeramente para hacer una serie de operaciones en el cual se pueden vizualizar una cantidad indefinida de registros.

Ahora bien, dicho cuestionario realiza 5 operaciones como tal, primero la del tipo 'Get'(Seleccionar) la cual trae todos los registros que estan ubicados en la BD. Luego tenemos la funcion 'INSERT' la cual inserta un nuevo registro a la BD mediante un formulario que funciona a traves de la absorcion de una API que posee dicho funcionamiento. La funcion 'UPDATE' actualiza un registro en caso de querer cambiar un dato. Ya finalizando tenemos la funcion 'DELETE' que lo que hace es eliminar un registro, en caso de ya no necesitarlo. Algo para agregar es que la funcion 'UPDATE' como 'DELETE' funciona con el tipo 'GET'.

Vuelos

El presente formulario "Vuelos" nos sirve primeramente para hacer una serie de operaciones en el cual se pueden vizualizar una cantidad indefinida de registros.

El formulario vuelos realiza 4 operaciones como tal, Cargar los datos, insertar, actualizar y eliminar esto para una cantidad indefinida de registros. Para esto usa los siguientes metodos: GET, POST, PUT Y DELETE.

Aviones

El formulario aviones tiene varias funciones. Registra el tipo de avión, las horas de vuelo realizadas, la cantidad de pasajeros que puede llevar, cuando fue su primer vuelo, donde fue construido y la cantidad de vuelos que ha realizado. Así se lleva un control de los diferentes tipos de avión.

En el formulario se pueden realizar cuatro operaciones; cargar toda la lista de registros que contiene la base de datos, editar un registro del formulario, eliminar un registro e insertar un nuevo registro. Para esto se cumple con los siguientes métodos: GET, PUT, DELETE Y POST.
